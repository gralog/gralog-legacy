This package defines classes that are especially useful to define Structure-Beans in GrALoG. To define a Structure-Bean in GrALoG you should subclass one of {@link de.hu.gralog.beans.support.DefaultPropertyChangeListenableBean} respectively {@link de.hu.gralog.beans.support.DefaultPropertyAndDisplayChangeListenableBean} depending on whether or not you want to fire DisplayChangeEvents. The interface {@link de.hu.gralog.beans.support.StructureBean} gives your bean access to the {@link de.hu.gralog.structure.Structure}-instance it belongs to.

Plugin-Developers

Please refer to the description of {@link de.hu.gralog.structure.StructureBeanFactory} if you want to implement your own Structure-Bean for GrALoG.
