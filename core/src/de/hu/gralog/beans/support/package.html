This package defines classes that are especially useful to define GraphBeans in Gralog. To define a GraphBean in Gralog you should extend one of {@link de.hu.gralog.beans.support.DefaultPropertyChangeListenableBean} respectively {@link de.hu.gralog.beans.support.DefaultPropertyAndDisplayChangeListenableBean} depending on whether or not you want to fire DisplayChangeEvents. The interface {@link de.hu.gralog.beans.support.GralogGraphBean} gives your bean the possibility to have access to the {@link de.hu.gralog.graph.GralogGraphSupport}-instance it is currently belonging to.

Plugin-Developers

Please refer to the description of {@link de.hu.gralog.graph.GraphBeanFactory} if you want to implement your own GraphBean for Gralog.
