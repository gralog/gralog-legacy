This package defines all {@link java.beans.PropertyEditor PropertyEditors} that are currently available in GrALoG.

{@link java.beans.PropertyEditor PropertyEditors} provide renderers and editors to GrALoG that are shown to the user when editing the corresponding properties. GrALoG is fully compatible to {@link java.beans.PropertyEditor} conventions and also provides an Extension to {@link java.beans.PropertyEditor} that allows to specify {@link java.awt.Component Components} as renderers {@link de.hu.gralog.beans.propertyeditor.PropertyEditorRendererExtension}.
