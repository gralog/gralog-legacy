This package defines all {@link java.beans.PropertyEditor PropertyEditors} that are currently avaible in Gralog. PropertyEditors provide renderers and editors to Gralog that are shown to the user when editing this Property. Gralog is fully compatible to {@link java.beans.PropertyEditor} conventions and also provides an Extension to {@link java.beans.PropertyEditor} that allows to specify {@link java.awt.Component Components} as Renderers {@see PropertyEditorRendererExtension}.
