This package defines all {@link java.beans.PropertyDescriptor PropertyDescriptors} that are currently avaible in Gralog.

{@link java.beans.PropertyDescriptor PropertyDescriptors} are used to define {@link java.beans.PropertyEditor PropertyEditors} for properties of algorithms, graphs, vertices and edges in Gralog. You can define your own PropertyDescriptor by either subclassing one of these or by directly subclassing {@link java.beans.PropertyDescriptor}. Note that you also have to define a corresponding {@link java.beans.PropertyEditor}.
