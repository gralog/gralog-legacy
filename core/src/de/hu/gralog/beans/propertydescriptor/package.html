This package defines all {@link java.beans.PropertyDescriptor PropertyDescriptors} that are currently avaible in Gralog. PropertyDescriptors are used to define {@link java.beans.PropertyEditor PropertyEditors} for properties that can be defined for algorithms, graphs, vertices and edges in Gralog. You can define your own PropertyDescriptor by either subclassing one of these or directly subclassing {@link java.beans.PropertyDescriptor}. Note that you also have to define a corresponding {@link java.beans.PropertyEditor}.
