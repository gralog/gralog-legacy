This package defines all classes needed to implement algorithms in GraLog. An Algorithm in Gralog is basically a JavaBean ( please refer to {@link de.hu.gralog.beans} for a general introduction to JavaBeans in Gralog and further references ) that implements {@link de.hu.gralog.algorithm.Algorithm}.

Being a JavaBean means that Gralog-Algorithms can specify their properties by using the JavaBeans-Framework that provides functionality to define {@link java.beans.Customizer Customizers}, {@link de.hu.gralog.beans.propertyeditor PropertyEditor} and {@link de.hu.gralog.beans.propertydescriptor PropertyDescriptions} for these properties respectively Algorithms. It basically means that Gralog-Algorithms can be hugely customized taking arbitrary Parameters and providing their own GUI-Components to edit these properties.

Plugin-Developers

Please see {@link de.hu.gralog.algorithm.Algorithm} for an example of how to implement your own Algorithm and {@link de.hu.gralog.beans} for a general introduction to Beans.
