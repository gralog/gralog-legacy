This package defines all classes needed to implement algorithms in GrALoG.

An algorithm in GrALoG is a JavaBean ( please refer to {@link de.hu.gralog.beans} for a general introduction to JavaBeans in GrALoG and further references ) that implements {@link de.hu.gralog.algorithm.Algorithm}.

Being a JavaBean means that GrALoG-Algorithms can specify their properties by using the JavaBeans-Framework that provides functionality to define {@link java.beans.Customizer Customizers}, {@link de.hu.gralog.beans.propertyeditor PropertyEditors} and {@link de.hu.gralog.beans.propertydescriptor PropertyDescriptors} for these properties respectively algorithms. It basically means that GrALoG-Algorithms can be hugely customized, take arbitrary parameters and provide their own GUI-Components to edit their properties.

Plugin-Developers

Please see {@link de.hu.gralog.algorithm.Algorithm} for an example of how to implement your own algorithm and {@link de.hu.gralog.beans} for a general introduction to JavaBeans.
