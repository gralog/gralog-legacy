This package defines the standard types for elements (vertices and edges) of a Gralog-Graph.

It also defines classes that should be used by Plugin-Developers in order to define their own Gralog-Vertices and -Edges. See {@link de.hu.gralog.graph.types.elements.DefaultListenableVertex} on how to define your own vertextype in Gralog and {@link de.hu.gralog.graph.types.elements.DefaultListenableEdge} for how to define your own edgetype in Gralog.
