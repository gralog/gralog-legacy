This package defines the standard types for elements (vertices and edges) of a GrALoG-Structure.

It also defines classes that should be used by Plugin-Developers in order to define their own GrALoG-Vertices and -Edges. See {@link de.hu.gralog.structure.types.elements.DefaultListenableVertex} on how to define your own vertex-type in GrALoG and {@link de.hu.gralog.structure.types.elements.DefaultListenableEdge} for how to define your own edgetype.
